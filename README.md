# Automated Internship & Project Portal

A real-world MVP for college internship and project management.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Framer Motion, Lucide React
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **Authentication**: JWT, BcryptJS

## Getting Started

### Prerequisites
- Node.js
- MongoDB (Running locally or Atlas)

### Backend Setup
1. `cd server`
2. `npm install`
3. Create `.env` file (copied from `.env.example` if available)
4. `npm start` or `node server.js`

### Frontend Setup
1. `cd client`
2. `npm install`
3. `npm run dev`

## Roles
- **Student**: Browse opportunities, apply, track status.
- **Company**: Post internships/projects, manage applicants.
- **Faculty**: Approve/Reject projects, monitor student progress.
- **Admin**: Overall system management.
